Dependency Manager CAOS

Documentation Caos: https://pypi.org/project/caos/

To run do the next steps:
install python 3

install CAPS:
  pip3 install caos
Setup the virtual environment:
  caos prepare
  caos update
Run the aplication:
  caos run
